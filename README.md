# Математика простым языком

Проект: Telegram-бот с AI-ассистентом для школьной математики.

## Описание

Этот продукт помогает ученикам решать задачи, объясняет ошибки и поддерживает тренировку по темам. В центре внимания — не просто готовый ответ, а мягкая подсказка, которая помогает ребёнку понять, где он ошибся.

Главная идея:

- ученик отправляет фото с решением;
- бот распознаёт рукописный текст;
- анализирует решение;
- даёт подсказку, а не готовый ответ;
- объясняет ошибку и предлагает похожие задачи;
- репетитор получает отчёты по ошибкам.

## Ключевые возможности MVP

- загрузка фото решения
- распознавание рукописного текста
- анализ ошибки
- Hint Mode — мягкие подсказки вместо ответа
- пошаговое объяснение
- подбор похожих задач
- базовая аналитика и отчёт для репетитора

## Стек технологий для первой версии

- Telegram bot: Python + aiogram
- Backend: FastAPI
- Web App: HTML/CSS/JavaScript
- База данных: PostgreSQL (или Supabase)
- AI: GPT-4o или GPT-4.1-mini
- Хостинг: Railway / Vercel

## Структура бота

### Главное меню

- Начать путь
- Выбрать тему
- Мой прогресс
- Помощь

### Выбор темы

- Алгебра
- Геометрия
- Логарифмы
- Степени
- Дроби
- Подготовка к экзамену

### Загрузка задачи

- Отправить решение
- Написать задачу
- Выбрать тему
- Подсказка по шагам

### Анализ решения

- Небольшая подсказка
- Показать объяснение
- Ещё одна задача
- Вернуться в меню

## Как начать

1. Развернуть Telegram-бота
2. Подключить backend и AI
3. Настроить хранение фото
4. Запустить первый сценарий анализа решения

## Цель

Создать полезный цифровой инструмент, который помогает ученикам учиться математике без страха, а репетитору — работать эффективнее.
